## 1.  Installing

#### Install dependencies

```bash
$ npm i
```

#### Installing pm2
```bash
$ npm i -g pm2
```

#### Environments

Copy `.env.example` to `.env` and set environments

##  2. Running

```bash
$ npm run dev
```

## 3. Building
* this following commands build and run project
#### Development

```bash
$ npm build:dev
```

#### Production


```bash
$ npm build
```
